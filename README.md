# Mini-Design-System-rhombix-technologies-task
Mini Design System including Colors, Status Colors, Typography, Buttons, Inputs, Cards, Spacing

This repository contains a **Mini Design System** created in Figma, designed to maintain **visual consistency, scalability, and usability** across UI projects. It includes essential design components such as colors,Status Colors, typography, buttons, input fields, cards, and spacing rules.

## Typography

| Name | Size | Weight | Use |
|------|------|-------|-----|
| H1 | 32px | Bold | Page Title |
| H2 | 24px | SemiBold | Section Title |
| H3 | 20px | Medium | Card / Component Title |
| Body Large | 16px | Regular | Main content |
| Body Small | 14px | Regular | Secondary text |
| Caption | 12px | Regular | Helper text / Meta info |
| Button | 14px | SemiBold | Buttons & Actions |
| Label | 14px | Medium | Form Labels |


## Buttons

**States:**
 Default: `#4F46E5` / White text
 Hover: `#4338CA` / White text
 Disabled: `#E5E7EB` / `#9CA3AF` text

**Height:** 44px  
**Radius:** 8px  
**Padding:** 12px vertical, 16px horizontal

 Created as **Figma components with variants** for easy reuse.


## Input Fields

**States:**
 Default: Border `#D1D5DB`
 Focused: Border `#4F46E5`
 Error: Border `#EF4444`

**Padding:** 12px  
**Radius:** 8px  
**Placeholder Color:** `#6B7280`
